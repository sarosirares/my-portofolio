# ATmega32 Microcontroller Projects

This repository contains various applications developed using the ATmega32 microcontroller.

The projects explore fundamental concepts such as digital input/output, timers, interrupts, and interfacing with external components. The goal is to build practical embedded systems and understand how microcontrollers interact with hardware.

To support development and testing, simulations were created using Proteus, allowing the projects to be designed and verified without physical hardware.

This is an ongoing collection of embedded systems experiments.
