Computer Architecture – Linux Practice

In this application I worked in a Linux environment (Debian/Slax) using the terminal. I practiced basic system commands and created several Bash scripts.

I used chmod to change file permissions and understand access rights. I wrote scripts that calculate sums and products, read user input, and work with command line arguments. I also used control structures like if, for, and case. (Simple stuff just to get used with this environment)

Additionally, I created scripts to find minimum/maximum values, ignore certain inputs, and measure execution time. I practiced simple networking commands (ifconfig, route) and communication between users using write.

Overall, this helped me understand basic Linux usage, scripting, and how low-level operations work in a system.
