# Elevator Automation – Ladder Logic (LogixPro)

This project implements an automated elevator control system using ladder logic, developed and tested in LogixPro.

The objective was to simulate the behavior of a real elevator system, including floor selection, movement control, and response to user inputs. The program manages elevator operations such as moving between floors, handling requests, and ensuring proper sequencing of actions.

## Features

* Floor selection using input buttons
* Controlled movement of the elevator (up/down)
* Logical decision-making based on current position and requested floor
* Basic safety and control conditions
* Sequential operation handling multiple requests

## Implementation

The system was designed using ladder logic, a graphical programming language commonly used in PLCs. The logic is structured to:

* Read inputs (floor buttons, sensors)
* Process conditions and determine direction
* Control outputs (motor direction, indicators)

The project was built and simulated in LogixPro, allowing testing of the elevator behavior without physical hardware.

## Learning Outcomes

* Understanding of PLC programming concepts
* Practical use of ladder logic for automation systems
* Implementation of sequential control processes
* Simulation and debugging using LogixPro

This project represents a practical introduction to industrial automation and control systems.
