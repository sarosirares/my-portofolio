# Webots Robot Sorting Project (Unfinished)

This project was developed using the Webots robotics simulator.

The goal was to create a robot that moves in a straight line while using an onboard camera to detect numbered blocks placed sequentially along its path. The intended behavior was for the robot to compare numbers on adjacent blocks and rearrange them into order.

Currently, the robot can:

* Move forward in a straight line
* Detect numbered blocks using its camera
* Compare values between consecutive blocks
* Swap two blocks when the next block has a smaller number than the current one

The project is incomplete and does not yet implement a full sorting algorithm or handle all edge cases.
